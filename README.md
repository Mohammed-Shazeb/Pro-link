# Pro Link

A social networking platform for professionals.

## Project Structure

- **Backend**: Node.js/Express API
- **Frontend**: Next.js React application

## Setup

### Backend
```bash
cd backend
npm install
node server.js
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
